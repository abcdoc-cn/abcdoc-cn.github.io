GRANT 权限

常用的系统权限集合：

  1. CONNECT： 基本的连接
  2. RESOURCE： 程序开发
  3. DBA： 数据库管理

常用的数据对象权限：

  1. ALL ON 数据对象名
  2. SELECT ON 数据对象名
  3. UPDATE ON 数据对象名
  4. DELETE ON 数据对象名
  5. INSERT ON 数据对象名
  6. ALTER ON 数据对象名

中文字符按拼音排序

  • 第一个参数是需要排序的字段
  • 第二个参数值可选下列字符串
    • SCHINESE_RADICAL_M 按照部首（第一顺序）、笔划（第二顺序）排序
    • SCHINESE_STROKE_M 按照笔划（第一顺序）、部首（第二顺序）排序
    • SCHINESE_PINYIN_M 按照拼音排序，系统的默认排序方式为拼音排序
