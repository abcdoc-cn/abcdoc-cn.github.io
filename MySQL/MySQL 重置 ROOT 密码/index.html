MySQL 重置 ROOT 密码

2021-04-26
MySQL
  • MySQL
  • 速查

Windows

  1. 打开 DOS 窗口，转到 mysql\bin 目录

  2. 输入 mysqld --skip-grant-tables 回车。如果没有出现提示信息，那就对了

  3. 新开一个 DOS 窗口，保留之前的 DOS 窗口

    1. 输入 mysql 回车，成功会出现 MySQL 提示符
  1. 注销系统，重新进入，可以使用新的密码登陆 mysql

Linux

  1. 先登陆服务器，找到自己的 my.cnf 文件

    1. 一般在 /etc 目录下
    2. 如果不在，可以用 find / -name my.cnf 命令找一下

  2. vi my.cnf 命令编辑该文件

    1. 在[mysqld]下面加上 skip-grant-tables，作用是登陆时跳过登陆认证

  1. 用 service mysqld restart 重启mysql
  2. 重置密码
  1. 删除 my.cnf 中添加的 skip-grant-tables
  2. 重启 mysql
